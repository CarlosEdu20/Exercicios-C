# Exercícios em C - Linguagem C Completa e Descomplicada

Este projeto contém exercícios resolvidos em C organizados por capítulos/temas, desenvolvidos durante meus estudos com o livro *Linguagem C Completa e Descomplicada*.

## Estrutura

- `capitulo_01_Operadores/`: Operadores aritméticos e lógicos (3 arquivos)
- `capitulo_02_Estruturas_Condicionais/`: Estruturas de decisão - if, else, switch (5 arquivos)
- `capitulo_03_Estruturas_repeticao/`: Loops - for, while, do-while (22 arquivos)
- `capitulo_04_Vetores_e_matrizes - arrays/`: Vetores e matrizes (20 arquivos)

## Como compilar

Este projeto usa CMake (padrão no CLion). Você pode rodar qualquer executável criado pelo `add_executable` no CMakeLists.txt.

### Compilação manual:
```bash
gcc -o output capitulo_01_Operadores/ExecOperadores123456.c
./output
```
